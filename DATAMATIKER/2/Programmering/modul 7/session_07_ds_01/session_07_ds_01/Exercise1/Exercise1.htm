Exercise 1

Write a program that computes the size of a directory structure including all nested sub directories.

This problem yields a recursive solution.

The class File from the Java API may be used. See more about File in the on line help.

You may want to use the following methods:

  • public File(String pathname)
  • public String getAbsolutePath()
  • public boolean isDirectory()
  • public long length()
  • public File[] listFiles()
